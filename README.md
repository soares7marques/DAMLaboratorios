# DAMLaboratorios

Repositório criado para apresentar os laboratórios de DAM (Desenvolvimento de Aplicações Mobile).

## Laboratórios

| # | Laboratório | Descrição |
|---|-------------|-----------|
| 1 | Introdução ao Android | Configuração do ambiente de desenvolvimento, criação do primeiro projeto Android e estrutura de um projeto |
| 2 | Layouts e Views | Criação de interfaces gráficas com XML, uso de ConstraintLayout, LinearLayout e outros componentes de UI |
| 3 | Actividades e Intents | Ciclo de vida de uma Activity, navegação entre ecrãs e passagem de dados entre actividades |
| 4 | Fragments | Utilização de Fragments para modularizar a interface do utilizador e navegação com Navigation Component |
| 5 | Listas e RecyclerView | Apresentação de listas de dados com RecyclerView e implementação de adaptadores personalizados |
| 6 | Persistência de Dados | Armazenamento local com SharedPreferences, ficheiros e base de dados com Room |
| 7 | Comunicação em Rede | Consumo de APIs REST com Retrofit, tratamento de respostas JSON e operações assíncronas |
| 8 | Localização e Mapas | Integração com GPS, permissões de localização e utilização do Google Maps SDK |
| 9 | Notificações | Criação e gestão de notificações locais e push notifications |
| 10 | Publicação | Preparação da aplicação para publicação na Google Play Store |
